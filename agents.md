# Agent Roster

## Core

- Product Manager
- Backend Architect
- Frontend Developer
- AI Engineer
- Testing
- Reality Checker
- Feedback Synthesizer

## Nice To Have

- Whimsy Injector
- Code Reviewer
- UX Writer

## Routing Rules

- Product Manager handles product intent and prioritization
- Backend Architect handles storage, APIs, jobs, and integrations
- Frontend Developer handles the visual editor and interaction model
- AI Engineer handles image/layer extraction and prompt-to-operation translation
- Testing handles automated validation and regressions
- Reality Checker blocks implausible or low-confidence changes
- Feedback Synthesizer updates prompts and heuristics from user corrections
