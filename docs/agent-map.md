# Agent Map

This map connects common user requests to default owners.

## Product Manager
- scope
- roadmap
- MVP
- market readiness
- release planning

## Backend Architect
- APIs
- database
- queues
- workers
- storage
- rollback

## AI Engineer
- layer extraction
- OCR
- prompt-to-action
- confidence scoring
- safety gating

## Testing
- TDD
- benchmarks
- regressions
- acceptance criteria

## Reality Checker
- plausibility
- safety
- low-confidence edits
- sanity checks

## Feedback Synthesizer
- learning loops
- user corrections
- prompt refinement
- heuristic updates

## Code Reviewer
- code quality
- architecture consistency
- spec compliance

## Default Routing Examples

- "make this faster" -> Backend Architect or AI Engineer
- "is this safe to ship" -> Testing + Reality Checker
- "make it market ready" -> Product Manager + Backend Architect
- "turn this into editable layers" -> AI Engineer
- "learn from user feedback" -> Feedback Synthesizer
